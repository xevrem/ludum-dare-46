# ludum-dare-46
- Theme: Keep it Alive

## Zombie Robot Horde!
Survive as long as you can, you only have 50 Health.

### How to Play:
- Move Up: W
- Move Left: A
- Move Right: D
- Move Down: W
- Drop Bomb: Space
- Set Trap: Shift
- Throw Elixer: Control

### Scoring:
- Zombies: 5 pts
- Robots: 10 pts
